# Global Minds · SignalBridge — Pitch Video

> Hybrid Remotion + Playwright workflow producing a 75s pitch video for
> investor meetings (Global Minds) as a guided tour through the 6 dashboard
> sections of the USDCOP trading system.

---

## Deliverables

| # | File | Audience | Variant |
|---|------|----------|---------|
| 1 | `out/pitch-raw-1080p.mp4` | Handoff / feedback | Raw (no music, no SFX, scene markers) |
| 2 | `out/pitch-final-1080p.mp4` | Final presentation | Full (music + SFX + transitions) |
| 3 | `out/pitch-final-vertical.mp4` | LinkedIn / mobile | 1080×1920 |

## Narrative structure (8 scenes, 75s)

1. **P01 · Hook** (5s) — "Febrero 2026. El mercado USD/COP cambió." + equity teaser
2. **P02 · Inicio** (8s) — Login → Hub with 6 módulos staggered reveal
3. **P03 · Dashboard** (15s) — **OOS 2025 validation** + backtest replay + `+25.63% · Sharpe 3.35 · p=0.006`
4. **P04 · Producción** (12s) — 2026 YTD + **regime gate blocked 13/14 weeks**
5. **P05 · Forecasting** (9s) — 9 modelos ML + 63 backtests walk-forward
6. **P06 · Análisis** (9s) — IA semanal GPT-4o · 16 semanas generadas
7. **P07 · SignalBridge** (12s) — Ejecución + Airflow DAGs + kill switch + AES-256
8. **P08 · Outro** (5s) — Global Minds · SignalBridge logo + disclaimer

## Asset pipeline

### Playwright dashboard captures

Located in `public/captures/*.webm`. Generated by:
```bash
npx tsx scripts/capture-dashboard.ts           # all 9 clips
npx tsx scripts/capture-dashboard.ts S03       # single clip
```

Pre-requisites: stack running on `:5000` (dashboard) + `:8080` (Airflow).
Credentials: `admin@trading.com` / `admin123`.

**Clips**: S01-login, S02-hub, S03-dashboard-scroll, S03-replay, S05-production-live,
S06-analysis-chat, S07-forecasting-zoo, S08-signalbridge, I04-airflow-dags.

### Procedural audio

Audio is **generated programmatically** (no external downloads, no licensing risk).
Generated by:
```bash
npx tsx scripts/generate-audio.ts
```

**Output**:
- `public/audio/music/pitch-ambient.wav` (75s stereo, chord progression Am → F → C → G)
- `public/audio/sfx/*.wav` (6 SFX: whoosh, tick, number-tick, impact, pop, riser)

All WAV files are written from raw Float32 → PCM16 math. Zero external dependencies.

## Audio licensing

All audio is **procedurally generated** in this repo via `scripts/generate-audio.ts`.
No downloads, no third-party samples, no attribution required. Every PCM sample is
computed from the formulas in the script (sine waves, filtered noise, ADSR envelopes).

Music: Am → F → C → G chord progression with slight stereo detune + vibrato + low-pass
smoothing. SFX: procedural synthesis (exponential decay, noise bursts, frequency sweeps).
